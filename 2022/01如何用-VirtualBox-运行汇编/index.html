如何用 VirtualBox 运行汇编

发表于 2022-01-22 更新于 2022-01-31

最近用到一些 x86 汇编，希望每次都可以用虚拟机从 Bootloader 阶段开始跑程序，这里记录一下如何在 Windows 环境下编译汇编源代码，并用 VirtualBox 运行汇编。

环境

  • VS Code
  • Nasm（需添加环境变量）
  • VirtualBox
  • FixVhdw

用 VirtualBox 运行

新建虚拟电脑

  1. 在 VBox 主界面菜单栏点击新建；
  2. 类型选择“Other”，版本保持“Dos”不变；
  3. 内存选择 32M 不变；
  4. 保持默认“现在创建虚拟硬盘”；
  5. 点击创建， 硬盘文件类型选择“VHD（虚拟硬盘）”；
  6. 存储方式选择固定大小；
  7. 硬盘大小设置为 16MB；
  8. 创建完成

此时启动应找不到 bootloader。
关闭时应选择强制退出。

使用 FixVhdw 写入虚拟机的硬盘

  1. 选择刚创建的虚拟硬盘（.vhd）文件；
  2. 数据文件 1 选择编译好的 .bin 文件;
  3. 起始 LBA 扇区号必须为 0 ；
  4. 写入（或者“写入并执行VBox虚拟机）；
  5. 启动虚拟机，显示运行结果。
